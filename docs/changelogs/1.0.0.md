## NerdKube 1.0.0 — first public release

First stable release of **NerdKube** for the *Nerds Quadrados* modpack (Minecraft 1.21.1, NeoForge).

### Highlights

- **Four-pillar endgame** after the End Remastered gate: Technology, Magic, Exploration, and Agriculture — each pillar yields a unique amulet for the final ritual.
- **Cube Maker ritual** — place the four amulets on themed pedestals, insert a vanilla Crafter, and craft the signed **`nerd_cube`** trophy (one per player).
- **Cross-mod progression** — tuned crafts and rituals across Oritech, PneumaticCraft, Occultism, Malum, Mystical Agriculture, Alex's Mobs, and more.
- **FTB Chunks integration** — automated machines (e.g. Oritech Laser Arm) break blocks using the placing player's UUID so claims are respected.
- **PT-BR first** — in-game UI and JEI categories in Brazilian Portuguese, with English lang support.

### Requirements

- Minecraft **1.21.1**, NeoForge **21.1.232+**, Java **21**
- Designed for the closed **Nerds Quadrados** pack; optional integrations degrade gracefully when a mod is absent.
- Does **not** replace the End Remastered 16-eye gate or FTB Quests chapter `os_olhos_do_fim`.
